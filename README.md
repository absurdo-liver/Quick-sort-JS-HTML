Quicksort Visualizer
====================

A simple, interactive web application that visualizes the Quicksort algorithm using HTML, CSS, and vanilla JavaScript.

Features
--------

*   **Algorithm Visualization:** Dynamically renders the Quicksort process step-by-step @ 10ms/step (adjustable in inspect element: `intervalTime`).
*   **Customization:** Adjust the array length or input your own custom array via a Comma-Separated Value (CSV) input.
*   **Interactive Controls:** Buttons to start the visualization, generate a new random array, and adjust settings.

Getting Started
---------------

### Prerequisites

You only need a modern web browser to run this application. No special server environment is required.

### Installation

No installation required.

### Usage

Simply navigate to https://absurdo-liver.github.io/Quick-sort-JS-HTML/ using your browser of choice.
